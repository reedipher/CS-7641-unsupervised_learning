- Clone repository
    git clone git@github.com:reedipher/CS7641_unsupervised_learning.git unsupervised_learning

- Install Anaconda python if not already installed
    https://www.anaconda.com/distribution/
    Download Linux Python 3.7 installer and follow instructions (bash Anaconda*.sh)

- Install latest scikit-learn
    $ pip install -U scikit-learn

- Go into directory and open up jupyter notebook
    $ cd unsupervised_learning/code
    $ jupyter notebook

- Open 'Assignment 3.ipynb'

- Click the 'Kernel' tab and choose to 'Restart and Run All'
